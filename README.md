# 红楼梦 Book Wiki

这是《红楼梦》的专题 Wiki，用于把原始章回文本整理成可阅读、可检索、可持续扩展的知识网络。

## 当前产物

- `chapters/`：120 回章节页，包含概要、人物、事件、地点、概念、伏笔和阅读重点。
- `texts/simplified/`：120 回简体原文，供阅读使用。
- `texts/traditional/`：120 回繁体原文，从 Raw 导入，含繁简互链。
- `characters/`：90 个人物页，含核心人物、红学家/批书人（胡适、周汝昌、俞平伯、蔡元培、王国维、脂砚斋等）。
- `events/`：30 个关键事件页，说明事件、章节、叙事作用和阅读抓手。
- `concepts/`：37 个主题概念页，含群芳碎、色空观念、神瑛侍者与绛珠仙草、葬花吟与芙蓉女儿诔、大观园诗社等专题概念。
- `maps/` 与 `timelines/`：人物、家族、大观园、全书时间线和宝黛钗线。
- `outputs/`：速读指南、人物手册、主题导读、阅读路线、编译总评。
- `redology/`：从人物、概念、事件和诗词切入的专题研究综述，用外部材料补强 Wiki 网络。

## 维护原则

1. 原始文本只读，不在本 Wiki 中直接改写原文。
2. 章节页负责“这一回发生什么”，不替代原文。
3. 人物、事件、概念、地点页必须提供跨章节理解，不能只是链接列表。
4. 红学争议、版本问题、作者归属统一放入 `redology/`，但必须从现有人物、概念、事件或诗词页回链进入。
5. 外部引用必须写入吸收该观点的具体页面 `sources:`，不能只在总页或最终回复里列链接。
6. 研究型扩展采用“现有节点补强 + 专题综述 + 双向回链”的方式，让知识网络越读越密。
7. 批量改动后必须更新 [[02_Learn/08_book-wikis/红楼梦/log.md]] 和 [[02_Learn/08_book-wikis/红楼梦/outputs/红楼梦编译总评.md]]。
8. 新页面若只有标题和空泛说明，不应创建；宁可先保留为纯文本提及。
