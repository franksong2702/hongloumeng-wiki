# 香怜

## 人物小传

香怜是贾家义学中的学生，与[[玉爱]]同属贾府远房支派的子弟。第九回中，他与玉爱的暧昧关系被[[金荣]]撞见并出言讥讽，成为学堂群殴事件的直接导火索。与玉爱一样，香怜在书中的存在感极低，几乎没有独立台词，他的叙事功能远大于其人物深度。

曹雪芹以"香怜"这个带有柔美、婉约色彩的名字暗示了这个人物的气质与命运。在义学这个本应书声琅琅的场所中，香怜与玉爱的暧昧关系不仅暴露了当时书院风气的败坏，更折射出贾府子弟教育的全面失败。一个贾家义学，竟成了子弟们"恋风流"的场所，这对于标榜诗礼传家的贾府来说无疑是一种莫大的讽刺。

香怜与玉爱的关系在整部小说中起到了一种"微缩映射"的作用：贾珍在宁国府的淫乱行为与义学中少年们的暧昧关系，本质上都是同一个家族道德堕落的不同表现。上自族长，下至子弟，无一能够逃脱。曹雪芹通过这样两个几乎没有台词的小人物，完成了对贾府道德状况的全景式批判。

## 关键关系

- 暧昧对象 → [[玉爱]]（义学中的亲密伙伴，第九回群殴事件的导火索）
- 冲突对象 → [[金荣]]（讥讽二人引发斗殴）
- 关联人物 → [[贾蔷]]（发现二人关系后示意挑拨）
- 叙事功能 → 义学腐败的缩影，与贾珍乱伦形成上下互文
