## 螃蟹咏原文

第038回菊花诗会之后，众人复又要了热蟹来，宝玉道："今日持螯赏桂，亦不可无诗。"三首螃蟹咏即席而成。回目"薛蘅芜讽和螃蟹咏"中"讽和"二字是理解宝钗诗的关键——讽刺性的唱和。

### 宝钗螃蟹咏

```
桂霭桐阴坐举觞，长安涎口盼重阳。
眼前道路无经纬，皮里春秋空黑黄。
酒未敌腥还用菊，性防积冷定须姜。
于今落釜成何益，月浦空余禾黍香。
```

### 宝玉螃蟹咏

```
持螯更喜桂阴凉，泼醋擂姜兴欲狂。
饕餮王孙应有酒，横行公子却无肠。
脐间积冷馋忘忌，指上沾腥洗尚香。
原为世人美口腹，坡仙曾笑一生忙。
```

### 黛玉螃蟹咏

```
铁甲长戈死未忘，堆盘色相喜先尝。
螯封嫩玉双双满，壳凸红脂块块香。
多肉更怜卿八足，助情谁劝我千觞。
对斯佳品酬佳节，桂拂清风菊带霜。
```

## 逐首精读

### 宝钗螃蟹咏——世情讽刺的巅峰

#### 原文

```
桂霭桐阴坐举觞，长安涎口盼重阳。
眼前道路无经纬，皮里春秋空黑黄。
酒未敌腥还用菊，性防积冷定须姜。
于今落釜成何益，月浦空余禾黍香。
```

#### 逐句解读

**第一联："桂霭桐阴坐举觞，长安涎口盼重阳。"**

"桂霭"写桂花香气如云霭笼罩，"桐阴"写出梧桐树荫下的清凉。起笔先造境——一个雅致的秋天宴饮场景。但第二句"长安涎口盼重阳"立刻把调子从清雅拉向俗世。"长安"指代京城、世间；"涎口"——让人流口水的，是螃蟹。京城里所有人盼着重阳吃螃蟹。"涎口"二字极为生动，写出世人贪嘴的欲望姿态。宝钗起笔雅，落脚俗，这种雅俗对照是她整首诗的基调。

**第二联："眼前道路无经纬，皮里春秋空黑黄。"**

全诗最核心的一联，也是[[02_Learn/08_book-wikis/红楼梦/poetry/螃蟹咏.md|螃蟹咏]]中最有名的两句。

"眼前道路无经纬"——螃蟹横行走路，没有方向感（"经纬"本指南北和东西方向，引申为规矩、秩序）。这是写螃蟹，但更是在写人世中那些横行霸道、不按常理出牌的人。他们不讲道理，不管方向，只顾横冲直撞。

"皮里春秋"——这是用典。《晋书·褚裒传》说褚裒"有讥评，但不显言，人莫能窥，号为'皮里阳秋'"。"阳秋"即"春秋"，因避简文帝太后阿春讳而改。意思是表面不露褒贬，但内心自有品评。"空黑黄"——蟹壳里面是蟹黄和蟹膏（黑色和黄色），但"皮里春秋"在这里一语双关：既指蟹肚中的黑黄之物（蟹膏、蟹黄），又暗讽那些心里自以为有"春秋笔法"的人，到头来肚中不过是些黑黄的蟹膏蟹黄罢了。所谓"空"——一切心机、算计、权谋，最后不过是空。

众人"叫绝"，宝玉说"写得痛快"，就是因为这两句的讽刺力度极大，而措辞又极典雅含蓄。讽刺而不谩骂，尖刻而不露骨，这是宝钗的笔力。

**第三联："酒未敌腥还用菊，性防积冷定须姜。"**

由讽刺转向实写——吃螃蟹的注意事项：酒还压不住腥味，所以要配菊花；螃蟹性寒，必须佐以生姜来防。这两句回到生活细节，看似平实，但与前文的"空"形成呼应——无论怎样精心料理，螃蟹终究不过是一道菜。宝钗的写法是"放—收—放"结构：讽刺放得很大，生活细节收回来，结尾再放。

**第四联："于今落釜成何益，月浦空余禾黍香。"**

"落釜"——落入锅中被煮。螃蟹再横行、再霸道，最终不过被投进锅里。"成何益"——有什么意义呢？"月浦空余禾黍香"——水月之畔只剩下稻谷禾黍的香气。"空"字再次出现，呼应上文的"空黑黄"。一切权谋、横行、算计，最终归于虚无，只有自然界的禾黍之香永恒。

这句与[[02_Learn/08_book-wikis/红楼梦/poetry/螃蟹咏.md|螃蟹咏概览]]中所说"繁华中的冷意"直接相关：宴饮越热闹，读者越能感到这种繁华终将被时间吞没。

#### 诗中暗示的作者性格和命运

宝钗这首诗显示了她三个核心特质：

1. **理性冷静**——她能从一个饮食游戏中抽身而出，以旁观者角度冷眼看世情。这与她在[[02_Learn/08_book-wikis/红楼梦/characters/薛宝钗.md|薛宝钗]]形象中一贯的"冷"相一致（冷香丸、任是无情也动人）。
2. **洞察力**——"皮里春秋空黑黄"一句显示她对人心权谋的理解极为透彻，但她选择"不显言"——这正是她一贯的处世策略。
3. **宿命感**——结尾"落釜""空余"透出她对一切繁华终将归于虚无的清醒认知。这种宿命感与她最终独守空房的命运形成对照。

脂砚斋评此诗"食螃蟹绝唱，小题目寓大意"，正说中了宝钗诗的核心特征——以极小之物写极大之题。

### 宝玉螃蟹咏——游戏人生的自我写照

#### 原文

```
持螯更喜桂阴凉，泼醋擂姜兴欲狂。
饕餮王孙应有酒，横行公子却无肠。
脐间积冷馋忘忌，指上沾腥洗尚香。
原为世人美口腹，坡仙曾笑一生忙。
```

#### 逐句解读

**第一联："持螯更喜桂阴凉，泼醋擂姜兴欲狂。"**

"持螯"——手持蟹螯。一开篇就进入吃的场景，与宝钗先造境再引螃蟹的从容不同，宝玉直截了当。"泼醋擂姜"——调料准备得兴师动众，"兴欲狂"三个字写出宝玉的投入和兴奋。他是真高兴，真的在享受这个宴席游戏。"兴欲狂"与宝钗的冷静形成鲜明对比。

**第二联："饕餮王孙应有酒，横行公子却无肠。"**

"饕餮王孙"——自指。"饕餮"是贪吃的怪兽，宝玉以"王孙"自居，自比为饕餮——他承认自己的贪嘴、贪玩、贪恋繁华。"横行公子"——螃蟹的别称。"无肠"既是蟹的生理特征（蟹确实没有明显的肠子），也暗指蟹没有心肠、没有思想。"应有酒"对"却无肠"，宝玉在自嘲——作为贵族公子，他应该饮酒作诗，但他面对的螃蟹（或者说他自己）却是"无肠"的。

**第三联："脐间积冷馋忘忌，指上沾腥洗尚香。"**

"脐间积冷"——蟹脐部位寒气最重，但"馋忘忌"——因为嘴馋连禁忌都忘了。宝玉的整个人生态度就在这四个字里——明知不该、不该、不应该，但因为"馋"（对世间美好的贪恋），一切禁忌都可以不顾。这正是宝玉的悲剧性性格。"指上沾腥洗尚香"——手上沾了蟹腥，洗过之后仍有香气。这句极为细腻，写出宝玉对感官体验的极度敏感和珍视——即便是"腥"，到他那里也能转化为"香"。

**第四联："原为世人美口腹，坡仙曾笑一生忙。"**

"坡仙"——苏轼。苏轼一生忙碌于仕途，曾经笑人"一生忙"。宝玉在这里说螃蟹原本就是为了满足世人口腹之欲而存在的，正如苏轼曾经笑人一生忙碌却无所得。这一联把螃蟹的命运扩大到了整个人生的层面，但宝玉的处理方式不是宝钗式的"空"，而是更轻松的"笑"——带着一种游戏、调侃、自我解嘲的态度。

#### 诗中暗示的作者性格和命运

宝玉这首诗显示了他的三个核心特质：

1. **游戏心态**——他不把螃蟹当讽世对象，而是当宴饮游戏的道具。"兴欲狂"是他的真实状态。
2. **感官主义**——"指上沾腥洗尚香"是他整个人格的缩影——他珍惜世间一切美的、香的、好的感官体验，哪怕这些体验本质上也是"腥"的、短暂的。
3. **自我嘲解**——"饕餮王孙"的自比、"馋忘忌"的坦白，都是他对自己性格弱点的清醒认知。他知道自己的弱点，但选择坦然接受而非改正。

宝玉的命运——他"馋"尽世间繁华，最终却"无肠"（无法在现实社会中有任何行动力），"洗尚香"——繁华散尽后只留下一丝记忆中的香气。

### 黛玉螃蟹咏——尖刻幽默与自我隐藏

#### 原文

```
铁甲长戈死未忘，堆盘色相喜先尝。
螯封嫩玉双双满，壳凸红脂块块香。
多肉更怜卿八足，助情谁劝我千觞。
对斯佳品酬佳节，桂拂清风菊带霜。
```

#### 逐句解读

**第一联："铁甲长戈死未忘，堆盘色相喜先尝。"**

"铁甲长戈"——把螃蟹的壳和螯比作披甲执戈的战士。这个比喻极为独特——别人写螃蟹写它的横行、写它的无肠，黛玉却写它的"武装到牙齿"。"死未忘"——即便被煮熟了装盘，甲壳和螯钳还在，仿佛到死都不忘战斗。这句暗含对螃蟹某种"敬意"——螃蟹是"战死"的。但第二句立刻转折——"堆盘色相喜先尝"——无论你怎么"铁甲长戈"，最后都被端上桌，我还是先尝为快。这种先敬后吃、先扬后抑的写法，显示了黛玉特有的幽默。

**第二联："螯封嫩玉双双满，壳凸红脂块块香。"**

这一联写螃蟹的"色相"——蟹螯里是白嫩的肉（"嫩玉"），蟹壳里是红红的膏脂。"双双满""块块香"——重叠词的使用，写出蟹肉的丰盛和美味。这一联极为感官，极为物质化，与黛玉平时"清高"的诗风形成反差。但正是这种反差让我们看到黛玉性格中的另一面——她不只是"葬花"的忧伤诗人，她也能写如此活泼、如此接地气的诗句。

**第三联："多肉更怜卿八足，助情谁劝我千觞。"**

"多肉更怜卿八足"——因为蟹肉多而喜爱它八只脚（"卿"是亲昵的称呼）。"助情谁劝我千觞"——谁能劝我喝下千杯酒？"助情"——螃蟹佐酒助兴。这句很关键——黛玉极少直接写自己的"情"和"饮"，这里却写"谁劝我千觞"，是一种极为罕见的自我敞开。在螃蟹咏的游戏氛围中，黛玉偶尔泄露了她内心深处的渴望——有人能劝她饮酒、与她同欢。

**第四联："对斯佳品酬佳节，桂拂清风菊带霜。"**

"对斯佳品酬佳节"——对着这样的美味来酬答佳节。"桂拂清风菊带霜"——桂花在清风中摇曳，菊花带着秋霜。结尾回到秋日的景物——桂花、清风、菊花、霜。这一联极其清雅，与前面"铁甲长戈"的粗犷和"嫩玉红脂"的感官形成鲜明的层次感。黛玉的结尾永远是"收"——从热闹中退出来，回到她最擅长的清冷之境。

#### 诗中暗示的作者性格和命运

黛玉这首诗显示了她三个核心特质：

1. **尖刻幽默**——"铁甲长戈死未忘"带着戏谑和调侃，与宝钗的讽刺不同，黛玉的幽默更轻盈、更带游戏性。
2. **感官与清高并存**——从"嫩玉红脂"的感官享受到"桂拂清风"的清冷超然，展现了黛玉性格的复杂性。
3. **隐藏的渴望**——"助情谁劝我千觞"是她内心深处的呼唤——没有人劝她的酒，没有人理解她的情。这正是黛玉在[[02_Learn/08_book-wikis/红楼梦/characters/林黛玉.md|林黛玉]]整体叙事中的命运——她永远是一个孤独的饮酒者。

## 三首对照

| 维度 | 宝钗 | 宝玉 | 黛玉 |
|------|------|------|------|
| **主题** | 世情讽刺 | 自我嘲解 | 尖刻幽默 |
| **视角** | 旁观者（冷眼） | 参与者（投入） | 戏谑者（调侃） |
| **关键词** | 空、空 | 兴欲狂、馋忘忌 | 死未忘、谁劝我 |
| **结尾基调** | 归于虚无 | 笑谈一切 | 退回清冷 |
| **对螃蟹态度** | 讽世工具 | 宴饮伙伴 | 被"吃"的战士 |
| **与菊花诗关系** | 脱离菊花诗的雅致，转向世俗讽刺 | 延续游戏精神 | 保留雅致但加入戏谑 |
| **风格关键词** | 理性、冷峻、沉著 | 投入、感官、自嘲 | 轻盈、幽默、清峭 |

### 风格差异与人物性格的关系

三首诗正好对应三个人物的核心性格结构：

**宝钗**的诗是一面"照世镜"——她看螃蟹不是看螃蟹，而是看螃蟹背后的人世。她的讽刺不针对个人，而是对整个社会秩序（"无经纬"）和人心（"皮里春秋"）的宏观判断。这与[[02_Learn/08_book-wikis/红楼梦/characters/薛宝钗.md|薛宝钗]]的"冷"和"识大体"一致——她永远是那个在热闹中保持清醒、在笑声中看见荒凉的人。

**宝玉**的诗是一面"自画像"——他写螃蟹其实是写自己。"饕餮王孙""横行公子""馋忘忌"——这些词既是写蟹也是写人。宝玉的"无肠"恰恰是他最核心的特征——他在现实世界中没有任何行动力、任何心机，但他对美有极其敏锐的感知力（"洗尚香"）。这与[[02_Learn/08_book-wikis/红楼梦/characters/贾宝玉.md|贾宝玉]]整体的"痴"和"情不情"一致。

**黛玉**的诗是一面"哈哈镜"——她把螃蟹写成披甲执戈的战士，带着戏谑和幽默。她不像宝钗那样严肃地讽世，也不像宝玉那样投入地享受，她以一种"居高临下"的姿态调侃螃蟹。"铁甲长戈死未忘"——这是黛玉特有的视角：在可笑的事物中看到某种庄重的悲剧性。这与[[02_Learn/08_book-wikis/红楼梦/characters/林黛玉.md|林黛玉]]整体的性格一致——她永远在"悲"与"谑"之间摇摆。

## 讽刺手法分析

### 螃蟹作为讽刺意象

螃蟹在中国古典文学中早有讽刺传统，但曹雪芹赋予了它新的层次：

1. **形态讽刺**——横行（无经纬）、无肠（无心肝）、铁甲长戈（虚张声势），螃蟹的身体特征被转化为对人性的讽刺。
2. **命运讽刺**——无论螃蟹怎样横行霸道、武装到牙齿，最终"落釜"（落入锅中）。这是对一切强横之徒的终极讽刺。
3. **世情讽刺**——"长安涎口盼重阳"——全京城的人都在盼着吃螃蟹。横行者和被横行者的关系在这里反转：被讽刺的对象（螃蟹）同时也被讽刺者（食客）所讽刺。

### "眼前道路无经纬"等名句的解读

"眼前道路无经纬"是整组螃蟹咏中最著名的一句，也是红学研究中讨论最多的一句。它的妙处在于三层含义：

1. **字面层**——螃蟹横着走，不走直线，没有方向感。
2. **世情层**——人世中那些不讲道理、不按规矩、横冲直撞的人。
3. **哲学层**——"经纬"不仅是空间方向，也是社会秩序、道德准则。"无经纬"暗示整个世界秩序的混乱和失范。

"皮里春秋空黑黄"同样有三层：

1. **字面层**——蟹壳里面是黑黄的蟹膏蟹黄。
2. **用典层**——"皮里春秋"出自《晋书》，指表面不露、内心有品评。
3. **讽刺层**——那些自以为有"春秋笔法"、内心有是非判断的人，到头来肚中不过是"黑黄"之物——所谓高尚的判断力，不过是些庸俗的欲望。

### 与第037回海棠诗社的对照（雅 vs 俗）

[[02_Learn/08_book-wikis/红楼梦/chapters/第037回.md|第037回]]海棠诗社的[[02_Learn/08_book-wikis/红楼梦/poetry/海棠诗.md|海棠诗]]与第038回螃蟹咏形成一组精妙的对照：

| 维度 | 海棠诗（第037回） | 螃蟹咏（第038回） |
|------|-------------------|-------------------|
| **咏物对象** | 海棠（花，高雅） | 螃蟹（食，世俗） |
| **情感基调** | 悲春伤秋，清雅忧伤 | 讽刺戏谑，世情洞察 |
| **社会性** | 内向的、个人的情感 | 外向的、社会的观察 |
| **女性角色** | 闺阁才情的展示 | 超越闺阁的世情批评 |
| **美学风格** | "偷来梨蕊三分白，借得梅花一缕魂" | "眼前道路无经纬，皮里春秋空黑黄" |

从海棠到螃蟹，曹雪芹完成了一个"从雅到俗"的叙事转换。这个转换不是降格，而是一种深化——大观园的女儿们不只是吟花弄月的闺阁诗人，她们有对现实世界的洞察力和批评能力。[[02_Learn/08_book-wikis/红楼梦/poetry/菊花诗.md|菊花诗]]和螃蟹咏放在同一回，正好构成"雅—俗"的完整谱系。

宝钗在菊花诗社中只作了《忆菊》《画菊》两首，风格"沉著"，但在螃蟹咏中却写出了最讽刺、最有锋芒的诗。这个反差本身就值得注意——宝钗的"冷"不仅表现在她的理性克制上，也表现在她对世情的冷眼旁观中。

## 阅读路线

1. **初读**：先读[[02_Learn/08_book-wikis/红楼梦/texts/simplified/第038回.md#^hlm-038-juhua-shi|第038回简体原文]]中螃蟹咏段落，感受三首诗的整体气质差异。
2. **精读**：结合本页面逐首解读，重点关注"眼前道路无经纬""皮里春秋空黑黄""馋忘忌""铁甲长戈死未忘"等核心句。
3. **对照**：将三首诗并列比较，理解宝钗的"讽"、宝玉的"戏"、黛玉的"谑"。
4. **拓展**：
   - 回顾[[02_Learn/08_book-wikis/红楼梦/chapters/第037回.md|第037回]]海棠诗社，理解"雅—俗"对照。
   - 对照阅读[[02_Learn/08_book-wikis/红楼梦/poetry/菊花诗.md|菊花诗]]，理解同回中两组诗的美学关系。
   - 参考[[02_Learn/08_book-wikis/红楼梦/redology/蔡义江.md|蔡义江]]、[[02_Learn/08_book-wikis/红楼梦/redology/欧丽娟.md|欧丽娟]]的红学研究，了解学术界对螃蟹咏的不同解读。
