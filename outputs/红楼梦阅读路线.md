# 红楼梦阅读路线

本路线将全书 120 回分为六个阶段，每阶段标注"读什么"和"关注什么"。如果你想要更详细的计划，请参考 [[02_Learn/08_book-wikis/红楼梦/outputs/红楼梦研究型阅读路线.md|红楼梦研究型阅读路线]]。

## 第一阶段：神话框架与贾府入场（第 001-010回）

- **结构功能**：全书的"设定层"——真假梦幻、太虚幻境、还泪前缘、四大家族关系网、家塾制度，都在这一阶段建立。
- **关注点**：
  - 第一回的"真假"结构如何奠定全书基调（甄士隐与贾雨村的双线叙事）
  - 第三回黛玉进贾府的空间描写（通过黛玉的眼睛认识贾府）
  - 第五回判词和红楼梦曲——这是全书的"命运总纲"，读完后四十回时会反复回看
  - 刘姥姥一进荣国府：外部视角的引入
- **概念入口**：[[02_Learn/08_book-wikis/红楼梦/concepts/真假.md|真假]]、[[02_Learn/08_book-wikis/红楼梦/concepts/梦.md|梦]]、[[02_Learn/08_book-wikis/红楼梦/concepts/还泪.md|还泪]]、[[02_Learn/08_book-wikis/红楼梦/concepts/谶语.md|谶语]]
- **入口**：[[02_Learn/08_book-wikis/红楼梦/chapters/第001回.md|第001回]] 至 [[02_Learn/08_book-wikis/红楼梦/chapters/第010回.md|第010回]]
- **研究延伸**：[[02_Learn/08_book-wikis/红楼梦/redology/第005回判词与红楼梦曲研究综述.md|第005回判词与红楼梦曲研究综述]]、[[02_Learn/08_book-wikis/红楼梦/redology/甄士隐贾雨村与真假结构研究综述.md|甄士隐贾雨村与真假结构研究综述]]

## 第二阶段：秦可卿之死与大观园成形（第 011-018 回）

- **结构功能**：宁府危机暴露 → 王熙凤掌权 → 元妃省亲 → 大观园落成。这是贾府从"内腐"到"外荣"的矛盾阶段。
- **关注点**：
  - 秦可卿之死中的"贾珍过度悲痛"——脂批暗示的原稿删改
  - 王熙凤协理宁国府的治理能力与权力滥用（铁槛寺弄权）
  - 第017-018回大观园建造与省亲——园林空间如何从礼仪工程变为青春世界
  - 省亲的奢华与元春的悲叹（"当日既送我到那不可见人的去处"）
- **概念入口**：[[02_Learn/08_book-wikis/红楼梦/concepts/园林.md|园林]]、[[02_Learn/08_book-wikis/红楼梦/concepts/权力.md|权力]]
- **入口**：[[02_Learn/08_book-wikis/红楼梦/chapters/第011回.md|第011回]] 至 [[02_Learn/08_book-wikis/红楼梦/chapters/第018回.md|第018回]]
- **研究延伸**：[[02_Learn/08_book-wikis/红楼梦/redology/秦可卿之死与宁府败相研究综述.md|秦可卿之死与宁府败相研究综述]]、[[02_Learn/08_book-wikis/红楼梦/redology/元妃省亲与贾府政治命运研究综述.md|元妃省亲与贾府政治命运研究综述]]

## 第三阶段：宝黛钗日常与青春世界展开（第 019-036 回）

- **结构功能**：全书最长的"日常段落"——宝黛情感深化、丫鬟群像展开、宝玉挨打的父权冲突。这是大观园青春世界的前奏。
- **关注点**：
  - 宝黛关系的"试探—和解—深化"三段式发展
  - 丫鬟群像的个体化：袭人的"规劝"、晴雯的"撕扇"、金钏的"投井"
  - 宝玉挨打：父权与"情"的正面冲突
  - 宝玉送旧帕：宝黛情感的最高峰
- **概念入口**：[[02_Learn/08_book-wikis/红楼梦/concepts/情.md|情]]、[[02_Learn/08_book-wikis/红楼梦/concepts/禄蠹.md|禄蠹]]、[[02_Learn/08_book-wikis/红楼梦/concepts/情友.md|情友]]
- **入口**：[[02_Learn/08_book-wikis/红楼梦/chapters/第019回.md|第019回]] 至 [[02_Learn/08_book-wikis/红楼梦/chapters/第036回.md|第036回]]
- **研究延伸**：[[02_Learn/08_book-wikis/红楼梦/redology/宝黛钗关系研究综述.md|宝黛钗关系研究综述]]、[[02_Learn/08_book-wikis/红楼梦/redology/宝玉挨打与仕途冲突研究综述.md|宝玉挨打与仕途冲突研究综述]]、[[02_Learn/08_book-wikis/红楼梦/redology/情意淫与宝玉精神结构研究综述.md|情意淫与宝玉精神结构研究综述]]

## 第四阶段：诗社、大观园日常与盛极之景（第 037-063 回）

- **结构功能**：大观园作为"青春共同体"的高峰期——海棠诗社成立、刘姥姥二进、群芳夜宴、元宵夜宴。这是全书最明亮也最脆弱的段落。
- **关注点**：
  - 海棠诗社的成立与诗词创作——人物才性与诗风的对应
  - 刘姥姥二进大观园：外部视角下的贾府奢华（"笑"背后的阶级差距）
  - 群芳夜宴的花签——每个人的花签就是她的命运预告
  - "盛极"中的暗流：迎春误嫁、香菱被夏金桂折磨
- **概念入口**：[[02_Learn/08_book-wikis/红楼梦/concepts/群芳碎.md|群芳碎]]、[[02_Learn/08_book-wikis/红楼梦/concepts/谶语.md|谶语]]、[[02_Learn/08_book-wikis/红楼梦/concepts/人情往来.md|人情往来]]
- **入口**：[[02_Learn/08_book-wikis/红楼梦/chapters/第037回.md|第037回]] 至 [[02_Learn/08_book-wikis/红楼梦/chapters/第063回.md|第063回]]

## 第五阶段：裂痕显现与大观园崩坏（第 064-080 回）

- **结构功能**：从尤二姐/尤三姐的悲剧到抄检大观园，从晴雯之死到芙蓉女儿诔，女儿世界被系统性摧毁。这是全书的转折核心。
- **关注点**：
  - 尤氏姐妹线：名分、误判和权力伤害
  - 抄检大观园：以礼法之名行破坏之实
  - 晴雯之死与芙蓉女儿诔——宝玉对"美之毁灭"的最深刻哀悼
  - 迎春误嫁、迎春之死：制度性暴力的受害者
  - 香菱病入膏肓：薄命主题的延续
- **概念入口**：[[02_Learn/08_book-wikis/红楼梦/concepts/礼法失序.md|礼法失序]]、[[02_Learn/08_book-wikis/红楼梦/concepts/家族衰败.md|家族衰败]]、[[02_Learn/08_book-wikis/红楼梦/concepts/薄命.md|薄命]]
- **入口**：[[02_Learn/08_book-wikis/红楼梦/chapters/第064回.md|第064回]] 至 [[02_Learn/08_book-wikis/红楼梦/chapters/第080回.md|第080回]]
- **研究延伸**：[[02_Learn/08_book-wikis/红楼梦/redology/大观园空间与抄检研究综述.md|大观园空间与抄检研究综述]]、[[02_Learn/08_book-wikis/红楼梦/redology/葬花吟与芙蓉女儿诔精读.md|葬花吟与芙蓉女儿诔精读]]、[[02_Learn/08_book-wikis/红楼梦/redology/婚姻制度与女性命运研究综述.md|婚姻制度与女性命运研究综述]]

## 第六阶段：后四十回的收束与败落（第 081-120 回）

- **结构功能**：宝玉婚姻、黛玉之死、贾府被抄、王熙凤败亡、宝玉出家。后四十回虽为续书，但在叙事上完成了"盛极而衰"的总结构。
- **关注点**：
  - 掉包计与黛玉焚稿——续书对"宝黛钗悲剧"的戏剧化处理
  - 元春之死、贾母之死——家族支柱的相继倒塌
  - 贾府被抄：外部清算
  - 宝玉中举与出家——从"灵"到"俗"再到"空"的回归
  - 刘姥姥三进荣国府：外部视角的收束（从"施"到"报"）
- **概念入口**：[[02_Learn/08_book-wikis/红楼梦/concepts/空.md|空]]、[[02_Learn/08_book-wikis/红楼梦/concepts/色空观念.md|色空观念]]
